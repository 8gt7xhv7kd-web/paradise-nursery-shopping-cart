# Paradise Nursery Shopping Cart

## Project Description
Paradise Nursery is an online houseplant shopping application built with React and Redux.
It allows users to browse houseplants by category, add them to a cart, and manage their purchases.

## Features
- Browse houseplants by category
- Add plants to shopping cart
- Update quantities and remove items
- Dynamic cart icon counter
- Responsive design

## Technologies Used
- React
- Redux Toolkit
- CSS3
- Vite

## Getting Started
```bash
npm install
npm run dev
```
